Z-Score Time Series Plot

This plot shows the standardized daily log returns (z-scores) over time. Reference lines indicate statistical significance levels: ±1σ (68%), ±2σ (95%), and ±3σ (99.7%).

Interpretation

Z-scores represent how many standard deviations each daily price change is from the mean. In a normal distribution, we would expect:

  • 68% of values within ±1σ
  • 95% of values within ±2σ
  • 99.7% of values within ±3σ

Financial markets typically show more extreme events (beyond ±3σ) than predicted by normal distribution theory, demonstrating the "fat tail" phenomenon described by Benoit Mandelbrot in "The Misbehaviour of Markets."
